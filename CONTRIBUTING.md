# Contributing to SSELexicon

Thank you for your interest in contributing!

## 🛠️ How to Contribute

- Fork this repository
- Clone your fork
- Create a new branch
- Make your changes
- Submit a Pull Request (PR)

## 📦 Build Requirements

- Visual Studio 2022+
- .NET 8.0 SDK
- Windows OS
